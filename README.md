# MyTailor – AI-Powered E-Tailoring Platform

## 📌 Project Overview
**MyTailor** is a modern AI-powered e-tailoring web application that helps users customize clothing online with smart measurement handling, tailoring management, and order tracking features.

The project is designed to simplify the tailoring process digitally by allowing customers to:
- Upload measurements
- Customize outfits
- Place tailoring orders online
- Track order progress

It also provides tailor/admin features for managing customers, orders, and product details efficiently.

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/mytailor.git
```

### 2️⃣ Go to Project Folder

```bash
cd mytailor
```

### 3️⃣ Install Frontend Dependencies

```bash
cd frontend
npm install
```

### 4️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

### 5️⃣ Setup Environment Variables

Create a `.env` file inside backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

---

## ▶️ Run Project

### Start Backend

```bash
cd backend
npm run dev
```

### Start Frontend

```bash
cd frontend
npm start
```
